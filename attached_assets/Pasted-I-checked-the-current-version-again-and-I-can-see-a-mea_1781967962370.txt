🚨 Remaining Issues

1. Statistics Still Don't Match



Top section:



307+ Properties

108+ Landlords



Bottom section:



10K+ Happy Tenants

850+ Landlords



These numbers still contradict each other. A potential investor or landlord will notice.



Recommendation



Use one source of truth from your database.



2. Your Product Positioning Is Confusing



This is currently your biggest issue.



You told me:



Users should not chat directly with landlords.



But the site still says:



Direct Contact

Direct WhatsApp Contact

"I contacted the landlord directly on WhatsApp..." in a testimonial.



If LIVAREX manages communication, then this messaging is working against your actual business model.



LIVAREX manages requests and inspections.



Right now you're describing both.



3. Too Many "Coming Soon" Cities



I still see:



Abuja

Port Harcourt

Kano

Ibadan

Enugu

Benin City



marked as Coming Soon.



This makes the platform feel smaller than it probably is.



Better Approach



Show only active markets.



For example:



Lagos

Ogun

Abuja



Then add more cities when listings exist.



4. Verification Needs More Transparency



Competitors are starting to show exactly how they verify users and properties. Some platforms explicitly mention NIN/BVN checks, identity verification, ownership checks, or fraud screening.



LIVAREX currently says:



Verified Landlords



But doesn't explain:



How

When

By whom

Add



A page:



How LIVAREX Verification Works



This could become one of your strongest trust features.



5. Missing "How It Works"



Users should immediately understand the process.



Example:



1. Browse verified properties

2. Request a property

3. LIVAREX verifies availability

4. Schedule inspection

5. Move in safely



Many successful property platforms make this flow obvious.



6. Homepage Feels Slightly Empty



You claim:



307+ Properties



But only a very small number are visible on the homepage.



Add

Recently Added

Featured Listings

Popular Properties



The homepage should feel alive.
